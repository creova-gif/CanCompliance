A full-stack compliance SaaS prototype for Canadian businesses. Single-file HTML app — no build step, no dependencies, no backend required to run.

## Moving to Production (Next.js)

When ready to productionize:

```
npx create-next-app@latest cancompliance-prod --typescript --tailwind --app
```

Then migrate each module to:
- `/app/api/check/[module]/route.ts` — server-side compliance logic
- `/lib/supabase.ts` — connect to Supabase project `ecuafonnvppsblxitgcu` (ca-central-1)
- `/components/modules/` — React components per module
